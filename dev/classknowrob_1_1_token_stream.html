◆ close()

void TokenStream::close ( )
virtual

Close the stream. This will push an EOS message, and all future attempts to push a non EOS message will cause an error. Once closed, a stream cannot be opened again. Note that a stream auto-closes once it has received EOS messages from all of its input channels.
